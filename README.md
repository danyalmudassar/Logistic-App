# 🚛 Logistics App Control Center

A professional, cost-effective logistics management prototype built with **React Native (Expo)** and optimized for **iOS, Android, and Web**.

**[Click here to view the Live Web Dashboard](https://your-vercel-link.vercel.app)** *(Replace with your actual Vercel URL)*

## 🚀 Key Features
- **Executive Dashboard:** Real-time tracking of shipments in transit, in process, and monthly stats.
- **Financial Health KPIs:** Automated calculation of Total Monthly LC (Letter of Credit) costs and the "Total Cost Gap."
- **Financial Oversight:** Mandatory "Finance Approval" gate for all new shipments.
- **Gap Analysis Engine:** Mandatory reasoning for any cost/time variance exceeding 5%, as per standard financial audit requirements.
- **Visual Status Tracking:** Dynamic step-progress bar showing the lifecycle from "Order Placed" to "Warehouse Received."
