# Logistics App

A cost-effective, cross-platform logistics application built with React Native (Expo) and Supabase.

## Architecture
- **Frontend:** React Native (Expo)
- **Backend/Database:** Supabase (PostgreSQL)
- **State Management:** React Context or Zustand
- **Navigation:** React Navigation (Expo Router)

## Development Workflow
- Use `npx expo start` to run the development server.
- Follow the Research -> Strategy -> Execution lifecycle for all tasks.
- Ensure all new features are accompanied by tests.

## Conventions
- Use functional components and hooks.
- Prefer Vanilla CSS (via StyleSheet) for styling.
- Maintain a clear separation between UI components and business logic.
